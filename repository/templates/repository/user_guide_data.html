User Guide

User Guide > Document Repository: DataLad

This guide will walk you through setting up DataLad, creating and managing datasets, and integrating them with IntegriLAB for streamlined tracking and updates.

Step 1: Download and Install DataLad
  • Visit the DataLad website or install via the command line:
    1. For Linux/macOS/Windows:
      2. pip install datalad
  • Ensure Git and Git-annex are installed as DataLad dependencies.
Step 2: Create a New Dataset Using DataLad
  • Open your terminal and navigate to the directory where you want to create the dataset.
  • Run the following command to create a new DataLad dataset:
    1. For Linux/macOS/Windows:
      2. datalad create my_dataset
  • This will initialize a dataset named my_dataset.
Step 3: Commit Changes to the Dataset
  • Add files to your dataset or make any changes.
  • Commit the changes with a meaningful message:
    datalad save -m "Added initial dataset files"
Step 4: Connect IntegriLAB with DataLad
  • Log in to your IntegriLAB account and select Data option.
  • Navigate to the Setup Connection section in the dashboard.
  • Enter your username, password, and server name when prompted.
  • Once authenticated, your DataLad repository will be linked to IntegriLAB.
Step 5: Add the DataLad Dataset to a Project
  • In the Data section, choose the Add New Repository option.
  • Associate the dataset with the corresponding project in IntegriLAB by using the path of your DataLad dataset.
Step 6: Update and Commit Changes
  • When there are updates to the dataset (e.g., new files or modifications), follow these steps:
  • To save updates:
    1. Go to the Data -> Home section in IntegriLAB and click on Commit Changes.
    2. Select the linked project and the related dataset
      3. to synchronize the latest version of the data with IntegriLAB.
    3. Click the Select Folder button, navigate to your dataset’s Git repository, enable hidden files, and select the .git/logs folder.

      4. Operating System Shortcut to Show Hidden Files
      Windows Press Ctrl + Shift + . in File Explorer
      Mac Press Command (⌘) + Shift + . in Finder
      Linux (Ubuntu, GNOME, etc.) Press Ctrl + H in File Manager
Repeat this step whenever significant updates are made to ensure all changes are documented.
