User Guide

User Guide > Document Repository: LabTrace

This guide will help you set up LabTrace, create and manage project, and integrate it with IntegriLAB to maintain traceability and certification of your project details.

Step 1: Create a LabTrace Account
  • Visit the LabTrace website and click CREATE ACCOUNT.
  • Fill in your details and follow the instructions to set up your account.
Step 2: Create a New Project
  • The Principal Investigator (PI) should create a new project within LabTrace.
  • Project members can request access to the project from the PI or the project owner.
Step 3: Connect LabTrace to IntegriLAB
  • Open IntegriLAB and navigate to the LabTrace section.
  • Select the Setup Connection option.
  • Add your LabTrace username and password to establish a connection with IntegriLAB.

  • Click on Commit Changes.
  • Choose the respective project to synchronize the latest version of the associated code, data, or document with IntegriLAB, generating a certificate of authenticity and traceability.
Repeat this step whenever significant updates are made to ensure all changes are documented.
