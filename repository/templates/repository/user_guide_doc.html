User Guide

User Guide > Document Repository: Overleaf

This guide will walk you through setting up Overleaf, creating and managing documents, and integrating them with IntegriLAB for streamlined tracking and updates.

Step 1: Create or Log In to Your Overleaf Account
  • Visit Overleaf.
  • If you already have an account, log in using your credentials.
  • If not, click Sign Up to create a new account.
Step 2: Create a New Project on Overleaf
  • After logging into Overleaf, click on the New Project button.
  • Choose a project type (e.g., Blank Project, Example Templates, or Upload Files).
  • Name your project and begin working on your LaTeX document.
Step 3: Enable Git Integration
  • Open your Overleaf project and click on the Menu (three horizontal lines in the upper-left corner).
  • Scroll down and find the Git option.
  • Generate a Git authentication token and save it securely.
  • Overleaf will also provide a Git URL for your project (e.g., https://git@git.overleaf.com/67446a7688c49dc7661cbd8c).
  • You'll use this token and URL to connect with IntegriLAB.
Step 4: Access the Document Repository in IntegriLAB
  • Log in to your IntegriLAB account.
  • Navigate to the Documents section from the dashboard.
  • This is where you will manage and monitor Overleaf projects linked to your IntegriLAB workspace.
Step 5: Connect IntegriLAB with Overleaf
  • In IntegriLAB, select Setup Connection option to link your Overleaf account.
  • Enter your Overleaf username and the Git authentication token (created in the previous step) to authorize the connection.
  • Once connected, your Overleaf projects will become accessible within IntegriLAB.
Step 6: Add an Overleaf Project to IntegriLAB
  • In the Documents section in IntegriLAB, click on Add New Repository option.
  • Enter the Overleaf project you want to associate with your current IntegriLAB project.
  • Enter the corresponding Git URL of the project.
  • This will enable you to manage and monitor updates to your Overleaf project directly within IntegriLAB.
Step 7: Commit Changes When Updates Are Made
  • As you work on your Overleaf document, any updates will need to be tracked in IntegriLAB.
  • To save updates:
    1. Go to the Documents -> Home section in IntegriLAB.
    2. Select the linked project and the related document to synchronize the latest version with IntegriLAB.
  • Repeat this step whenever significant updates are made to ensure all changes are documented.
