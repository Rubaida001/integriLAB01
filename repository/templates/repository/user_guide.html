Getting Started with IntegriLAB: Project Setup and Certification Steps
  • Tools Used: IntegriLAB integrates with GitHub, Overleaf, Datalad, and LabTrace for storing code, documents, data, and for certification purposes.
  • Account Setup: To use IntegriLAB, you need to set up your account with each of these systems.
  • Creating Projects: Once accounts are set up, create projects in these repositories to organize project-related tasks.
  • Connecting Repositories: Add these repositories to IntegriLAB to establish a connection with each project.
  • Tracking Commits:
    • When there is a new commit in the repository, you can add it to IntegriLAB.
    • If multiple commits occur in a single day, IntegriLAB will select only the most recent commit.
  • Certification Process:
    • Once you are satisfied with the current state of the project, navigate to the “Certification” option.
    • Certify the project state using blockchain to ensure it is notarized.
